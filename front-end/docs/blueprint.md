# **App Name**: CondoVote

## Core Features:

- User Authentication: Secure login for administrators, owners, and proxies with mandatory password change on first login.
- Role Management: Management of different roles of 'Administrador', 'Titular' and 'Procurador'.
- Proxy Management: Owners can manage their proxies: add, modify, enable/disable them.
- Voting System: System to support creating, running, and tallying votes based on lot ownership and type (lot vs house).
- Multiple Proxy Handling: Support for a single user acting as a proxy for multiple owners.
- Synchronous and Asynchronous Voting: Support both real-time (QR code access) and time-boxed voting sessions.
- Discussion Forum: Implement a threaded discussion forum for asynchronous voting topics.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trust and security.
- Background color: Light gray (#F0F0F5) to ensure clarity and focus on content.
- Accent color: Orange (#FF9800) to highlight key actions and calls to action.
- Body and headline font: 'PT Sans', a humanist sans-serif for clear readability and a modern, but friendly feel.
- Simple, modern icons to represent actions and options, ensuring ease of understanding, particularly on mobile devices.
- Clean, intuitive layout optimized for both desktop and mobile, with a focus on simplicity for end users.
- Subtle animations for vote confirmations and transitions, providing clear feedback without overwhelming the user.
